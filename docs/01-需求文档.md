# FLOWTEXT 项目需求文档

## 项目概述
FLOWTEXT 是一款 Windows 平台的 Markdown 编辑器，类似 Typora 的实时渲染风格，但提供可视化的文字格式修改工具栏，整体设计采用 Apple 风格的简洁清爽 UI。

## 技术选型
| 项目 | 选择 |
|------|------|
| 框架 | Electron |
| 目标平台 | Windows（优先） |
| 版本控制 | Git + GitHub 公开仓库 |

## 核心功能

### 1. Markdown 编辑器
- 实时渲染模式（Typora 风格）— 输入 Markdown 语法即时看到效果
- 支持的语法：标题、粗体、斜体、删除线、代码块、引用、有序/无序列表、表格、链接、分割线
- 支持快捷键操作

### 2. 可视化格式工具栏 ✨（核心差异化）
- 选中文字后弹出浮动工具栏
- 工具栏按钮（Apple 设计风格）：
  - 粗体 / 斜体 / 下划线 / 删除线
  - 字体颜色 / 背景高亮色
  - 字体大小
  - 标题级别（H1-H6）

### 3. 图片支持
- 拖拽插入图片
- 粘贴插入图片
- 文件选择对话框导入本地图片
- 图片在编辑器内渲染显示

### 4. 文件操作
- 新建文件
- 打开 .md 文件
- 保存 / 另存为 .md 文件

### 5. 导出功能
- 导出为 PDF
- 导出为 HTML

### 6. 界面风格（Apple 风格）
- 简洁清爽，大量留白
- 浅色/白色主题（未来可扩展暗色模式）
- 圆角设计，细腻阴影
- 无干扰的编辑界面
- macOS 风格标题栏

## 阶段性开发计划

| 阶段 | 内容 |
|------|------|
| Phase 0 | 项目初始化：目录结构、Git/GitHub、文档体系 |
| Phase 1 | Electron 应用骨架：能启动一个空白窗口 |
| Phase 2 | Markdown 编辑器核心：实时渲染 |
| Phase 3 | 浮动格式工具栏：可视化修改格式 |
| Phase 4 | 图片支持：拖拽/粘贴/对话框 |
| Phase 5 | 文件操作：新建/打开/保存 |
| Phase 6 | 导出功能：PDF + HTML |
| Phase 7 | 界面打磨：Apple 风格全面美化 |

## 非功能性需求
- 启动速度：冷启动 < 3秒
- 编辑流畅度：输入无卡顿
- 文件编码：统一 UTF-8
- 图片管理：支持相对路径和绝对路径
