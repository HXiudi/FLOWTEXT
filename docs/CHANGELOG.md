# FLOWTEXT 版本发布记录

## v1.1.0 (2026-06-02)

### 新增
- 🎨 **LOGO 图标** — FT 字母组合，Apple 风格
- 📦 **便携版 .exe** — 单文件，双击即用，约 45MB
- 🖼 **图片完全内嵌** — 粘贴/拖拽图片自动转为 base64，存于 .md 文件内

### 变更
- 🧹 **目录精简** — 删除无用文件和文件夹，结构清晰
- 🪟 **红绿灯圆点** — Windows 下隐藏，保留 macOS 代码
- ⚡ **移除 highlight.js** — 暂不需要代码高亮，后续按需添加

### 修复
- 图片保存不再依赖本地文件系统，纯内嵌方案更稳定
