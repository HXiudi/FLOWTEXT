# FLOWTEXT 设计规范（Apple 风格）

## 设计理念
- **简洁**：去掉多余装饰，内容即界面
- **清晰**：排版层次分明，信息一目了然
- **清爽**：大量留白，柔和配色

## 配色方案

### 主色调
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | #FFFFFF | 主编辑区纯白 |
| 侧栏/工具栏背景 | #F5F5F7 | Apple 经典的浅灰 |
| 分隔线 | #D2D2D7 | Apple 风格细线 |
| 主文字 | #1D1D1F | Apple 深灰文字 |
| 次要文字 | #86868B | 辅助信息 |
| 选中状态 | #007AFF | Apple 蓝 |
| 悬浮高亮 | #E8E8ED | 轻微灰色悬浮 |

### 功能色
| 用途 | 色值 |
|------|------|
| 粗体 | #1D1D1F |
| 红色 | #FF3B30 |
| 橙色 | #FF9500 |
| 黄色 | #FFCC00 |
| 绿色 | #34C759 |
| 蓝色 | #007AFF |
| 紫色 | #AF52DE |

## 字体
| 用途 | 字体族 |
|------|--------|
| 编辑器正文 | `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif` |
| 代码块 | `"SF Mono", "Cascadia Code", "Fira Code", "Consolas", monospace` |

## 尺寸规范

| 元素 | 尺寸 |
|------|------|
| 基础字号 | 16px |
| 标题 H1 | 32px / Bold |
| 标题 H2 | 24px / Bold |
| 标题 H3 | 20px / Semibold |
| 标题 H4 | 18px / Semibold |
| 行高 | 1.6 |
| 按钮圆角 | 6px |
| 面板圆角 | 10px |
| 工具栏圆角 | 12px |

## 间距
| 用途 | 间距 |
|------|------|
| 编辑器边距 | 40px 左右，60px 上下 |
| 段落间距 | 16px |
| 工具栏项目间距 | 8px |
| 图标与文字间距 | 6px |

## 阴影
| 层级 | 阴影值 |
|------|--------|
| 浮动工具栏 | `0 4px 12px rgba(0,0,0,0.12)` |
| 模态框 | `0 8px 30px rgba(0,0,0,0.2)` |
| 按钮悬浮 | `0 2px 6px rgba(0,0,0,0.1)` |

## 动效
| 场景 | 效果 |
|------|------|
| 工具栏弹出 | opacity 0 → 1 + translateY(-4px)，0.2s ease |
| 工具栏隐藏 | opacity 1 → 0，0.15s ease |
| 按钮 hover | 背景色 0.15s ease 过渡 |
| 页面切换 | 无过度动效（保持简洁） |
