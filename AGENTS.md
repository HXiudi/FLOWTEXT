# FLOWTEXT — AI 助手工作说明书

## 项目概述
FLOWTEXT 是一款 Windows 平台的 Markdown 编辑器（类似 Typora），采用 Electron 开发，Apple 风格 UI，核心特色是可视化的浮动格式工具栏。

## 开发日志
- 日志目录：`logs/`
- 命名规范：`YYYY-MM-DD.md`
- 每次启动工作时自动创建当日日志
- 日志包含：今日完成、待办事项、备注

## 工作模式

### 开发流程
1. 开始工作前先读取当日日志，了解进度
2. 每步操作前读取 `docs/04-开发步骤.md` 确认当前阶段
3. 每完成一个小步骤记录到日志
4. 每步完成后验证功能是否正常
5. 验证通过后进行 Git commit

### 执行规范
- 严格按照 `docs/04-开发步骤.md` 中的 Phase 顺序执行
- 不跳过任何步骤
- 每个步骤完成后必须验证
- 验证失败则回退修复，不推进下一步
- 设计决策优先参考 `docs/03-设计规范.md`

### 与用户沟通
- 用户决策时，提供选项并给出推荐
- 遇到技术难题时，解释原因并提供备选方案
- 每完成一个 Phase 后，向用户汇报进展
- 需要用户参与的操作（如提供 GitHub URL）及时询问
